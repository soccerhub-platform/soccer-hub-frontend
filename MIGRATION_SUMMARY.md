# Migration Summary

Документ обновлен под текущее состояние продукта и MVP-фокус по issue #1.

## Что зафиксировано в этой итерации

- Обновлена навигация, чтобы убрать скрытые рабочие маршруты:
  - Dispatcher menu: добавлены `Клиенты` и `Пробные`.
  - Admin menu: добавлены `Платежи` и `Сотрудники`.
- Явно разведена ответственность между `dispatcher/admins` и `admin/users`:
  - `dispatcher/admins` — создание администраторов и доступ к филиалам;
  - `admin/users` — операционное управление сотрудниками филиала.
- Обновлена архитектурная документация (`ARCHITECTURE.md`) под фактические модули и маршруты.

## Актуальная карта модулей

- `/dispatcher/*` — super admin / диспетчерские процессы
- `/admin/*` — процессы админа филиала (лиды, договоры, платежи, группы, тренеры)
- `/coach/*` — мобильный интерфейс тренера

## Актуальные команды проверки

- `npm run lint`
- `npm run typecheck`
- `npm test`
- `npm run build`

## Результат для MVP P0

Продуктовая навигация приведена к рабочему состоянию: ключевые сущности денежного цикла и операционной работы доступны из меню по ролям, а документация синхронизирована с текущим кодом.
